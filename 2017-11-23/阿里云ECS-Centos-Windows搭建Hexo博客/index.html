生成博客文章

使用 hexo n <文章名称>来新建文章，该命令会成成一个.md文件放置在 sources/_posts文件夹。
你可以使用其它软件编辑，或直接使用vim编辑

编辑完毕后，使用hexo g将.md文件渲染成静态文件，然后启动服务器hexo -server

现在就可以使用浏览器访问 http://localhost:4000查看博客了。

新建配置文件blog.conf

把/var/www/html/blog/目录授权给 git 用户。否则在同步部署时 git 用户没有权限，不容易部署成功。

注意：修改Nginx配置文件后、需要重启服务systemctl start nginx、而且阿里云ECS也需要开启相应的端口访问
