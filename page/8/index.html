Windows下安装 Python 虚拟环境 Virtualenv

发表于 2017-11-20 | 分类于 Python ， 环境搭建 | 阅读次数
前言

由于 Python 的版本众多，还有Python2和Python3的争论，因此有些软件包或第三方库就容易出现版本不兼容的问题。

通过 virtualenv 这个工具，就可以构建一系列 虚拟的Python环境 ，然后在每个环境中安装需要的软件包(配合 pip 使用)，这一系列的环境是相互隔离的。作为一个独立的环境就不容易出现版本问题，还方便部署。
