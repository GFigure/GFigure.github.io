Wego

Github: https://github.com/schachmat/wego

命令行里的天气预报。

安装使用：

  • wego 是用 Go 编写的，需要安装 Go 语言、然后再安装 wego。 Go 语言安装链接

第一次运行 weg 会生成一个配置文件（~/.wegorc），需要指定一个天气 API key。 你可以从 darksky.net获取一个免费的 API key。

成功取得 API key 之后，将它写入到 wego 配置中了。 打开 ~/.wegorc 文件， 找到这几个选项， 然后按照规则替换即可：

只需要配置这三个选项就够了。 forecast-lang 是天气预报数据的语言，wego 默认会把它设置成 en、改成 zh、 显示中文。

显示出来的天气信息包括：温度、风速和风向、可视距离、降水量和降水概率

默认情况下会显示 3 天的天气预报。如果要进行修改，可以通过参数改变天气范围（最多5天），比如要查看 5 天的天气预报：$ wego 5
