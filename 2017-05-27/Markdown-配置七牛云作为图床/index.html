Markdown-配置七牛云作为图床

发表于 2017-05-27 | 分类于 Hexo | | 阅读次数

Hexo在使用Markdown编辑发布博客时，经常要上传本地图片，渲染加载图片的效果很不理想。推荐使用云平台作为图床，提高编辑效率和图片效果。

在这里推荐使用七牛云，身份认证完成后，免费额度高：

注册配置七牛云

注册地址

注册完成之后，在资源主页中的对象存储里添加对象，如添加image，但不能为Bucket

注意选择的地区可能会影响到图片能否加载成功，如果出错修改地区重试。

简单的上传图片，复制外链

如果每次都要进入七牛云网页上传图片，有点麻烦，下面推荐使用插件快速上传图片

在chrome安装qiniu upload files插件

安装成功后打开，显示如下界面 :

配置qiniu-upload-files插件，进入七牛云个人中心的密钥管理，获取AccessKey和SecretKey

在建立的存储对象中的空间概览里获取测试域名，bucket指的是之前添加的存储对象名称image

配置完成后返回，将本地图片拖拽到此窗口即可上传文件，复制外链。

提示：若上传的图片占的空间大，最好要在线压缩后再上传。避免网站访问量大，请求的次数多，额度消耗的太快。
